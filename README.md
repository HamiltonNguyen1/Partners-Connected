# Partners-Connected
Partners Connected is a program that allows any school to store and collect information about any business or community partners, being able to store information about the partner's name, type, description, email, phone number, resources, and website if applicable. The information is then shown to the user(s) on the screen, where the user(s) can search and filter the partners as needed. Additional features include a Help Menu Chat Bot, which helps user(s) navigate the program if they are confused.
# Features
- Filter & Search Partners
- Add & Remove Partners
- Help Menu Chat Bot with Instructions
# System Requirements
- Windows 10
- Any Python IDE
- Internet (Only for Chat Bot & Websites)
